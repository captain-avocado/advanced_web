# [Gulp-Webpack] project template

## В проекте используется менеджер пакетов ```yarn```

### Установка:

    1. git clone https://github.com/klv-codehub/gulp-webpack-template

    2. cd gulp-webpack-template

    3. yarn
    
### Документация

```yarn start```

Запускает проект в режиме разработки.
